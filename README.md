# Kandidaten-Website – GitHub Pages

Dies ist eine **fertige, einseitige Website** für deinen Wahlkampf. Du kannst sie in wenigen Minuten über **GitHub Pages** veröffentlichen.

## Schnellstart (GitHub Pages)

1. Lade die Dateien dieses Ordners herunter (oder nutze das bereitgestellte ZIP).
2. Lege auf GitHub ein neues Repository an, z. B. `kandidaten-website`.
3. Lade alle Dateien (`index.html`, Ordner `assets/`) ins Repo (Root-Verzeichnis).
4. Gehe im Repo auf **Settings → Pages**.
5. Stelle bei **Build and deployment** ein:  
   - Source: **Deploy from a branch**  
   - Branch: **main** / **root**  
6. Speichern. Nach 1–2 Minuten ist deine Seite erreichbar unter:  
   `https://DEIN_USERNAME.github.io/kandidaten-website/`

## Anpassen

- Profilfoto ersetzen: `assets/portrait-placeholder.svg` durch `portrait.jpg` ersetzen und im HTML die `background`-URL anpassen.
- E-Mail, Instagram-Benutzername, Texte direkt in `index.html` anpassen.
- Impressum/Datenschutz im Footer ergänzen (gesetzliche Pflichtangaben in Deutschland!).

## Custom Domain (optional)

1. Domain bei Provider kaufen (z. B. `johannesschweiger.de`).  
2. In GitHub **Settings → Pages → Custom domain**: Domain eintragen.  
3. Beim Domain-Provider CNAME-Eintrag auf `DEIN_USERNAME.github.io` setzen.  
4. In diesem Repo eine Datei `CNAME` mit dem Domainnamen anlegen.

Viel Erfolg!
